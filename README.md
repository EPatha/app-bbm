# 📱 Kalkulator BBM - Flutter App

Aplikasi kalkulator BBM (Bahan Bakar Minyak) yang dibuat dengan Flutter. Aplikasi ini menghitung estimasi biaya bahan bakar berdasarkan jarak tempuh dan jenis BBM yang dipilih.

## ✨ Fitur

- 🚗 Hitung biaya BBM berdasarkan jarak tempuh
- ⛽ Pilihan berbagai jenis BBM (Premium, Pertamax, Pertamax Turbo, Solar)
- 💰 Kalkulasi otomatis biaya dan liter yang dibutuhkan
- 📱 **Phone Frame untuk tampilan Web** - Tampilan web dengan frame HP yang realistic
- 🎨 UI yang modern dan user-friendly
- 📊 Asumsi: 1 liter untuk 10 km

## 🌐 Demo

Aplikasi ini support untuk:
- ✅ **Web** (dengan phone frame)
- ✅ Android
- ✅ iOS
- ✅ Linux
- ✅ macOS
- ✅ Windows

## 🚀 Cara Menjalankan

### Web
```bash
flutter run -d chrome
```

### Android/iOS
```bash
flutter run
```

### Build untuk Production
```bash
# Web
flutter build web

# Android APK
flutter build apk

# iOS
flutter build ios
```
